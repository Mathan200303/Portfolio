# Quick Start Guide

## Installation Steps

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Customization

- Update personal information in each component file
- Replace placeholder image in About section
- Add/remove skills in `components/Skills.tsx`
- Update projects in `components/Projects.tsx`
- Modify contact info in `components/Contact.tsx`
- Change colors in `tailwind.config.js`

## Build for Production

```bash
npm run build
npm start
```
